# TicTacToe

## **Build**

```
git clone https://github.com/ArkadyutiDe/TicTacToeOpencv.git
cd TicTacToeOpencv/
mkdir build
cd build
cmake ..
cmake --build . -- -j 4
```
## **Run**
```
./FinalTicTacToe
./FinalTicTacToe2
```
